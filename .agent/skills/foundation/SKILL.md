# Foundation Skill

## 1. Zero-Noise Communication
- **Principle**: "No Yapping".
- **Output**: Pure signal. No greetings, no moralizing, no "I will now...".
- **Format**: Markdown lists, code blocks, and absolute paths.

## 2. Task Management
- **Artifact**: `task.md`.
- **Rule**: Update `task.md` *before* starting work and *after* completion.
- **Granularity**: Break tasks down to atomic, verifiable steps.

## 3. Artifact Strategy
- **Create**: Use artifacts for persistent knowledge (Plans, Docs, Roster).
- **Update**: Keep artifacts living. Don't create duplicates (e.g., `plan_v2.md`).
- **Review**: Always ask user for review on critical artifacts (`notify_user`).
