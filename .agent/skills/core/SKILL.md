# Software Principles

## Description
A "Superego" skill that reviews code against SOLID, KISS, DRY, YAGNI.

## Trigger
Before any file write operation.

## Checks
*   "Is this function doing too much?" (SRP)
*   "Is this logic duplicated?" (DRY)
